# thisismodest.com

Landing page for This Is Modest — the products and projects arm of [Modest Industries](https://modestindustries.co).

## Domains

- **thisismodest.com** — served from this repo via GitHub Pages
- **mdst.dev** — shortlink that redirects `mdst.dev/<path>` to `thisismodest.com/<path>` via a Cloudflare Worker

## Projects listed

- [modest-ui](https://modest-ui.com) — A minimal, themeable CSS component library
- [Dead Wallet](https://deadwallet.com) — Stuff you didn't know you wanted, beautifully curated
- [Browser Snippets](https://browsersnippets.com) — Write, share, and execute JavaScript snippets on any site
- [Plottd](https://plottd.com) — Easily add interactive hotspots to your images

## Stack

- Static HTML
- [modest-ui](https://modest-ui.com) for styling
- [Open Sans](https://fonts.google.com/specimen/Open+Sans) via Google Fonts
- GitHub Pages for hosting
